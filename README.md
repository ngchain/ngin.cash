# Index for NGIN

Domain https://ngin.sh

## Ready

Get `hugo` from [gohugo.io](https://gohugo.io/)

## Build

```bash
hugo -d docs
```

## Serve

On serve mode, editing the pages with WYSIWYG.

```bash
hugo serve -d docs
```

## Public

After build, upload the public folder contents to github page or VPS.
